# LIVING-SYSTEMS: Biology and Cognitive Science

## Premise

Living systems provide models for persistence, evolution, cognition, and self-organization. These concepts inform how Synapse thinks about the vault as a commons and the fabric as an evolving organism.

---

## Biology

### Autopoiesis (Maturana & Varela)

Self-producing systems: the system produces and maintains itself.

Criteria:
1. The system has boundaries
2. The system has internal processes
3. The system replaces its components over time
4. The system is the cause of itself

**Synapse:** The fabric produces new fabric. The vault grows. The engine is itself a blob.

### Ecosystem Models

- **Commons:** Shared resource, governed by dynamics
- **Succession:** Ecosystem stages toward climax
- **Keystone species:** Widely-referenced blobs whose removal cascades
- **Niche partitioning:** Multi-tenant manifests

### Immune System

Detects and eliminates "non-self."

**Synapse:** Adversarial test authorship (f_9). Governance gate as immune response.

### Germline vs. Soma

- **Germline:** Genetic line that persists (governance expression)
- **Soma:** Individual bodies that execute (executing blobs)

**Synapse:** Governance (germline) vs. invocation (soma).

---

## Cognitive Science

### Dual Process Theory (Kahneman)

- **System 1:** Fast, intuitive, automatic
- **System 2:** Slow, deliberate, analytical

**Synapse:** Direct invoke (fast) vs. council review (slow, deliberate).

### Embodied / Situated Cognition

Mind is not just in brain — it extends into environment and body.

**Synapse:** Node context = embodiment. Fabric as extended mind.

### Extended Mind (Clark & Chalmers)

Tools become cognitive artifacts. The mind includes what it can reliably access.

**Synapse:** Mesh as cognitive infrastructure. Telemetry as externalized memory.

### Predictive Processing (Friston)

Brain as prediction machine. Minimizes "surprise" (prediction error).

**Synapse:** Fitness function as Bayesian belief update. Telemetry as prediction error signal.

### Distributed Cognition

Mind across multiple agents and artifacts.

**Synapse:** Mesh of nodes. Council convergence.

### Metacognition

Thinking about thinking.

**Synapse:** Telemetry = introspection. Fitness function = self-evaluation.

## Key Insights

### The Vault as Commons

The vault is shared infrastructure. Like any commons, it can be degraded. Governance prevents tragedy.

**Question:** What are the "keystone blobs" — the widely-referenced infrastructure whose removal would cascade?

### Selection Pressure

Evolution requires:
1. Variation (mutation / new blobs)
2. Selection (governance)
3. Inheritance (reference to parent)

**Synapse:** Blob creation (variation) + fitness evaluation (selection) + content-addressing (inheritance).

### Memory and Identity

Biological systems have:
- Short-term memory (invocation context)
- Long-term memory (vault)
- Identity persistence (stable IDs)

**Synapse:** Context (ephemeral) + vault (persistent) + stable object IDs (identity).

## Open Questions

1. **Keystone blobs?** What blobs are so widely referenced that they cannot be deprecated?

2. **Selection pressure calibration?** How do we tune the fitness threshold to avoid both stagnation and chaos?

3. **Auto-poietic closure?** Can the fabric reach a state of self-sustaining evolution without external intervention?
